# 🍅 Pomodoro Timer

## 📌 Overview

A productivity timer built with Python and Tkinter that follows the Pomodoro Technique. The application alternates between focused work sessions and breaks while displaying progress through a simple graphical interface.

## ✨ Features

- ⏱️ 25-minute work sessions
- ☕ Short and long break sessions
- ▶️ Start and Reset buttons
- ✅ Progress tracking with check marks
- 🖥️ Interactive GUI using Tkinter

## 🛠️ Technologies Used

- Python 3
- Tkinter

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/satyam-katara/pomodoro-timer.git
```

2. Navigate to the project folder:

```bash
cd pomodoro-timer
```

3. Run the application:

```bash
python main.py
```
